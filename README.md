# PurePDF
Sebuah tools ringan untuk memanajemen file PDF tanpa harus mengupload file ke server apapun.
Semua proses terjadi dan dikerjakan di perangkat lokal user.
File Anda aman, tidak akan tersimpan di server manapun.
Karena ini 100% offline, maka prosesnya juga akan sangat cepat dan dapat digunakan meski tanpa internet.
